## Configuration

Copy the example environment file and fill in your values:

```bash

```

```dotenv
# .env.example

# Application
NODE_ENV=development
PORT=3000

# Database (PostgreSQL via Prisma)
DATABASE_URL="postgresql://user:password@localhost:5432/superproject_db"

# Authentication
JWT_SECRET=your-super-secret-jwt-key
JWT_EXPIRES_IN=15m
JWT_REFRESH_SECRET=your-refresh-secret
JWT_REFRESH_EXPIRES_IN=7d

# Optional: Redis for session caching

```

Run database migrations after configuring:

```bash
npx prisma migrate dev --name init
npx prisma generate
```

## Usage

### Development

```bash
# Start the development server with hot reload
npm run dev
```

The API will be available at `http://localhost:3000` and the React dev server at `http://localhost:3001`.

### Production

```bash
# Build the frontend bundle
npm run build

# Start the production server
npm start
```

### Quick API Example

```javascript
const axios = require('axios');

// Authenticate
const { data: auth } = await axios.post('http://localhost:3000/api/auth/login', {
  email: 'admin@example.com',
  password: 'secret',
});

// Use the token
const { data: users } = await axios.get('http://localhost:3000/api/users', {
  headers: { Authorization: `Bearer ${auth.token}` },
});

console.log(users);
// [{ id: 1, name: 'Alice', email: 'alice@example.com' }, ...]
```

## API Reference

### Authentication

#### `POST /api/auth/register`

Register a new user account.

**Request Body:**

```json
{
  "name": "Alice Smith",
  "email": "alice@example.com",
  "password": "securepassword123"
}
```

**Response `201 Created`:**

```json
{
  "id": 42,
  "name": "Alice Smith",
  "email": "alice@example.com",
  "createdAt": "2024-01-15T10:30:00.000Z"
}
```

---

#### `POST /api/auth/login`

Authenticate and receive a JWT token pair.

**Request Body:**

```json
{
  "email": "alice@example.com",
  "password": "securepassword123"
}
```

**Response `200 OK`:**

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refreshToken": "dGhpcyBpcyBhIHJlZnJlc2ggdG9rZW4...",
  "expiresIn": 900
}
```

### Users

#### `GET /api/users`

Retrieve a paginated list of users. Requires authentication.

**Headers:**

```
Authorization: Bearer <token>
```

**Query Parameters:**

| Parameter | Type   | Default | Description              |
|-----------|--------|---------|--------------------------|
| `page`    | number | `1`     | Page number              |
| `limit`   | number | `20`    | Results per page (max 100)|
| `search`  | string | вЂ”       | Filter by name or email  |

**Response `200 OK`:**

```json
{
  "data": [
    { "id": 1, "name": "Alice Smith", "email": "alice@example.com" },
    { "id": 2, "name": "Bob Jones",   "email": "bob@example.com"   }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 142,
    "pages": 8
  }
}
```

---

#### `GET /api/users/:id`

Retrieve a single user by ID.

**Response `200 OK`:**

```json
{
  "id": 1,
  "name": "Alice Smith",
  "email": "alice@example.com",
  "role": "admin",
  "createdAt": "2024-01-10T08:00:00.000Z",
  "updatedAt": "2024-01-14T12:45:00.000Z"
}
```

**Response `404 Not Found`:**

```json
{ "error": "User not found" }
```
